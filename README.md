## 静的WebサイトをGitHub Pagesで公開するまでの手順

１．ブラウザでGitHubu上にリポジトリを新規作成

２．ローカルの対象フォルダ上で、　git clone リポジトリURL

　　2.5．ページを編集
 
３．<b><font color="red">トップフォルダ直下に移動し</font></b>、　git add .

４．git commit -m "コメント"

５．git push
